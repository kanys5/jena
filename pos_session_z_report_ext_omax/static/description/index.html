POS Session Z Report

Overview

This Odoo app help to print pos session z report with opening and closing balance, gross sales, taxes, discount, net sales, Product and Variant wise sales bifurcation, category wise sales bifurcation, tax bifurcation, pricelist bifurcation, payment method bifurcation, Cash In & Cash Out details, Put Money In & Take Money Out Details,.
User can configure Show Details in Session Z Report to print in pos session z report.

Feature

  • Session report with opening and closing balance, gross sales, taxes, discount, net sales
  • Session report with Product and Product Variants wise sales.
  • Session report with bifurcation of category wise sales
  • Session report with bifurcation of taxes
  • Session report with bifurcation of pricelist
  • Session report with bifurcation of payment methods
  • Session report with Cash In & Cash Out details
  • Session report with Put Money In & Take Money Out details
  • User can easily print pos session z report from session record as well as from the pos screen directly based on the pos config.

    • It's optional to include & remove the below details from pos session z report.
    * Product Or Variant Wise Detail
    * Category Wise Sales
    * Taxes Detail
    * Pricelist Detail
    * Payment Detail
    * Cash In Out Details

How It Works


Session Z Report : User can print session report from Session form view [Button and print menu].



Session list view : User can print any numbers of session report from list view.



POS -> Configuration -> Setting : Configuration for allow to print Session Z Report directly from POS Screen.



POS Screen : From here user can print Session Z Report.



It's optional to include & remove the below details from pos session z report.
* Product Or Variant Wise Detail
* Category Wise Sales
* Taxes Detail
* Pricelist Detail
* Payment Detail
* Cash In Out Details

Session Z Report in PDF format.
